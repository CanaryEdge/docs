## 1. Get Your API Key

Sign up at [canaryedge.com](https://canaryedge.com) and grab your API key from the dashboard.

## 2. Send a Detection Request

<CodeGroup>
```bash cURL
curl -X POST https://api.canaryedge.com/v1/timeseries/entire/detect \
  -H "Content-Type: application/json" \
  -H "Ocp-Apim-Subscription-Key: YOUR_API_KEY" \
  -d '{
    "series": [
      {"timestamp": "2026-01-01T00:00:00Z", "value": 10.5},
      {"timestamp": "2026-01-01T01:00:00Z", "value": 11.2},
      {"timestamp": "2026-01-01T02:00:00Z", "value": 10.8},
      {"timestamp": "2026-01-01T03:00:00Z", "value": 45.0},
      {"timestamp": "2026-01-01T04:00:00Z", "value": 10.1}
    ],
    "granularity": "hourly",
    "sensitivity": 85
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.canaryedge.com/v1/timeseries/entire/detect",
    headers={
        "Content-Type": "application/json",
        "Ocp-Apim-Subscription-Key": "YOUR_API_KEY",
    },
    json={
        "series": [
            {"timestamp": "2026-01-01T00:00:00Z", "value": 10.5},
            {"timestamp": "2026-01-01T01:00:00Z", "value": 11.2},
            {"timestamp": "2026-01-01T02:00:00Z", "value": 10.8},
            {"timestamp": "2026-01-01T03:00:00Z", "value": 45.0},
            {"timestamp": "2026-01-01T04:00:00Z", "value": 10.1},
        ],
        "granularity": "hourly",
        "sensitivity": 85,
    },
)

print(response.json())
```

```javascript Node.js
const response = await fetch(
  "https://api.canaryedge.com/v1/timeseries/entire/detect",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "Ocp-Apim-Subscription-Key": "YOUR_API_KEY",
    },
    body: JSON.stringify({
      series: [
        { timestamp: "2026-01-01T00:00:00Z", value: 10.5 },
        { timestamp: "2026-01-01T01:00:00Z", value: 11.2 },
        { timestamp: "2026-01-01T02:00:00Z", value: 10.8 },
        { timestamp: "2026-01-01T03:00:00Z", value: 45.0 },
        { timestamp: "2026-01-01T04:00:00Z", value: 10.1 },
      ],
      granularity: "hourly",
      sensitivity: 85,
    }),
  }
);

console.log(await response.json());
```
</CodeGroup>

## 3. Read the Response

```json
{
  "isAnomaly": [false, false, false, true, false],
  "isPositiveAnomaly": [false, false, false, true, false],
  "isNegativeAnomaly": [false, false, false, false, false],
  "expectedValues": [10.5, 10.83, 10.65, 10.48, 10.55],
  "upperMargins": [2.1, 2.17, 2.13, 2.1, 2.11],
  "lowerMargins": [2.1, 2.17, 2.13, 2.1, 2.11],
  "period": 0
}
```

Point 4 (`value: 45.0`) is flagged as an anomaly — the model expected ~10.48 but observed 45.0, well outside the margin.
