## List Models

Retrieve all fine-tuned models for your organization.

<ResponseField name="models" type="array">
  Array of trained model records.
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.canaryedge.com/v1/models \
  -H "Ocp-Apim-Subscription-Key: YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "models": [
    {
      "machine_id": "pump-assembly-1",
      "created_at": "2026-03-10T14:30:00Z",
      "training_points": 500,
      "final_loss": 0.0012
    },
    {
      "machine_id": "conveyor-belt-3",
      "created_at": "2026-03-12T09:15:00Z",
      "training_points": 1200,
      "final_loss": 0.0008
    }
  ]
}
```
</ResponseExample>

## Delete Model

Remove a fine-tuned model. Future detection for this machine reverts to the generic model.

```bash
curl -X DELETE https://api.canaryedge.com/v1/models/pump-assembly-1 \
  -H "Ocp-Apim-Subscription-Key: YOUR_API_KEY"
```
