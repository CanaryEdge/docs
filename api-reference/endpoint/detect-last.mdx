---
title: "Detect Last Point"
api: "POST /v1/timeseries/last/detect"
description: "Check if the latest data point is anomalous"
---

Determines whether the most recent point in a series is anomalous, using prior points as context. Ideal for real-time streaming detection.

<ParamField body="series" type="array" required>
  Array of data points. The last point is evaluated; prior points provide context. Minimum 12 points.
</ParamField>

<ParamField body="granularity" type="string" required>
  Time interval between points.
</ParamField>

<ParamField body="sensitivity" type="integer" default="85">
  Detection sensitivity (0-99).
</ParamField>

<ResponseField name="isAnomaly" type="boolean">
  Whether the last point is anomalous.
</ResponseField>

<ResponseField name="isPositiveAnomaly" type="boolean">
  Whether the last point is a positive anomaly.
</ResponseField>

<ResponseField name="isNegativeAnomaly" type="boolean">
  Whether the last point is a negative anomaly.
</ResponseField>

<ResponseField name="expectedValue" type="number">
  The model's predicted value for the last point.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.canaryedge.com/v1/timeseries/last/detect \
  -H "Content-Type: application/json" \
  -H "Ocp-Apim-Subscription-Key: YOUR_API_KEY" \
  -d '{
    "series": [
      {"timestamp": "2026-01-01T00:00:00Z", "value": 10.5},
      {"timestamp": "2026-01-01T01:00:00Z", "value": 11.2},
      {"timestamp": "2026-01-01T02:00:00Z", "value": 45.0}
    ],
    "granularity": "hourly",
    "sensitivity": 85
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "isAnomaly": true,
  "isPositiveAnomaly": true,
  "isNegativeAnomaly": false,
  "expectedValue": 10.67,
  "upperMargin": 2.13,
  "lowerMargin": 2.13
}
```
</ResponseExample>
