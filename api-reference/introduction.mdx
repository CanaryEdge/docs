## Base URL

```
https://api.canaryedge.com
```

## Endpoint Groups

### Azure-Compatible Endpoints

Drop-in replacements for the Azure Anomaly Detector API. These endpoints use the same request/response schema as Azure v1.0 and v1.1:

- `POST /anomalydetector/{version}/timeseries/entire/detect` -- Batch anomaly detection
- `POST /anomalydetector/{version}/timeseries/last/detect` -- Last-point detection
- `POST /anomalydetector/{version}/timeseries/changepoint/detect` -- Changepoint detection

### Native API

Canary-specific endpoints with regime classification, energy scores, and machine management:

- `POST /v1/detect` -- Full detection with regime classification
- `POST /v1/baseline` -- Create/update machine baseline
- `GET /v1/baseline` -- Retrieve machine baseline
- `GET /v1/status/{machine_id}` -- Machine status
- `GET /v1/machines` -- List machines
- `DELETE /v1/machines/{machine_id}` -- Delete machine
- `GET /v1/usage` -- Usage statistics

### System

- `GET /health` -- Health check (no auth required)

## Authentication

All endpoints (except `/health`) require an API key. See [Authentication](/api-reference/authentication) for details.

```bash
curl https://api.canaryedge.com/anomalydetector/v1.1/timeseries/entire/detect \
  -H "Ocp-Apim-Subscription-Key: YOUR_API_KEY"
```

## Rate Limits

Rate limiting uses a token bucket algorithm keyed by organization ID.

| Parameter | Value |
|-----------|-------|
| Default rate | 100 requests/second per org |
| Bucket refill | Continuous |
| Burst capacity | 100 tokens |

When the rate limit is exceeded, the API returns HTTP 429:

```json
{
  "code": "TooManyRequests",
  "message": "Rate limit exceeded"
}
```

## Errors

All error responses follow the Azure error schema:

```json
{
  "code": "InvalidSeries",
  "message": "The 'series' field must have at least 12 points."
}
```

| Status | Code | Condition |
|--------|------|-----------|
| 400 | `InvalidSeries` | Series too short (<12), too long (>8640), unsorted, or has duplicates |
| 400 | `InvalidModelArgument` | `maxAnomalyRatio` or `sensitivity` out of range |
| 401 | `Unauthorized` | Missing or invalid API key |
| 404 | `NotFound` | Machine not found (no baseline) |
| 429 | `TooManyRequests` | Rate limit exceeded |
| 500 | `InternalServerError` | Model inference failure |
